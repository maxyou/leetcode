给定一棵二叉树，我们知道删除树上一条边可以使树分成两部分，现在请决定删除哪一条边，可以使得两部分的和最接近，并返回两部分和的最小差的绝对值。

示例 1：

输入：[3,5,8,2,4]
输出：0
解释：
删除 3 - 5 的边，两部分的和均为 11，最小差值为0。

示例 2：

输入：[3, -1]
输出：4
解释：
删除 3 - (-1) 的边，两部分的和为 3 和 -1，最小差值为4。
